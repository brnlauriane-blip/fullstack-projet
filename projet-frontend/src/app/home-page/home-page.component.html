Home Page

This web application allows the HR to create, update and delete users or job positions. In each section, users and user types, a list and a form can be found. The users list allows getting an overview of employees, names, emails, job positions, and their job IDs. In this section, it is possible to update and delete employee profiles. The users form allows creating new employee profiles. The user types list allows getting an overview of job positions in the company. Each job position is related to a job ID. In this section, it is possible to update or delete job positions. The user types form allows creating new job positions.

Below you will find links to the user lists and the option to create a user under “Users,” or you can access the list of user types or the user type creation form.
